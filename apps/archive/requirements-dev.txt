# ==================== NOTES ====================
#
# Quick Commands:
# - Run all tests: pytest
# - Run with coverage: pytest --cov
# - Run specific test: pytest apps/archive/tests/unit/test_file.py
# - Run marked tests: pytest -m unit
# - Parallel execution: pytest -n auto
# - Watch mode: ptw  (if pytest-watch installed)
#
# Coverage Report:
# - Terminal: pytest --cov --cov-report=term-missing
# - HTML: pytest --cov --cov-report=html
# - Open report: open htmlcov/index.html
#
# Code Quality:
# - Format code: black apps/
# - Sort imports: isort apps/
# - Lint code: flake8 apps/
# - Type check: mypy apps/
#
# Performance:
# - Profile tests: pytest --profile
# - Slow tests: pytest --durations=10
